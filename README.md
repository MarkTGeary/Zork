A Zork-based Prison Escape Game Coded in Java.

Requirements:
To run the code ensure that you have:
Java installed (version 17+)
JavaFx installed



How To Play:
Move around the prison gathering objects and interacting with NPCs to try and escape the prison and get back to the pub!
Certain items may be needed to enter different rooms or unlock them.

Features:
Connected Rooms which the user can move between
The user can either type commands or use buttons for the main commands of the game.
Note that the top text box is for typing commands and the bottom text box prints the output of the game as the game plays out.
A save and load game feature using serialisation.
Interactive Objects that help the user escape.
Interactive NPCs which can help or arrest the user along the way.





How To Win Game:
Collect Poison from chef, then poison the guard in the yard. Take the shovel in the storage room and dig up gold in the yard. Take the gold to Bob and trade
it for a guard uniform. You will now be able to enter southern part of prison. Grab Key from Infirmary and use it to unlock the warden's office. Take the access code
from the warden's office and use it to unlock the prison exit. Enter the car in the exit and drive. You will arrive at the pub where you win when you drink the beer!
